# Installation

## Stable release

To install pyutils, run this command in your terminal:

```sh
uv add pyutils
```

Or if you prefer to use `pip`:

```sh
pip install pyutils
```

## From source

The source files for pyutils can be downloaded from the [Github repo](https://github.com/baskettaz/pyutils).

You can either clone the public repository:

```sh
git clone https://github.com/baskettaz/pyutils
```

Or download the [tarball](https://github.com/baskettaz/pyutils/tarball/main):

```sh
curl -OJL https://github.com/baskettaz/pyutils/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd pyutils
uv sync
```
